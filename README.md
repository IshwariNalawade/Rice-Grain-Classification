# Rice Grain Classification

This project classifies rice grain varieties using image processing and machine learning.

## Technologies

- Python
- OpenCV
- NumPy
- Pandas
- Scikit-learn

## Workflow

- Image preprocessing
- Feature extraction
- KNN Classification
- Model Evaluation

## Results

- Accuracy: 85%
- Confusion Matrix
- Precision
- Recall

## Dataset

Kaggle Rice Grain Dataset
